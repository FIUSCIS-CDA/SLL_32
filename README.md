#### MODULE NAME
       SLL_32

#### DEPENDENCIES
       MUX32
       Decoder_32

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     32
H   |      5

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     32     

#### DESCRIPTION
       Takes  a 32-bit input A, and shifts it left by H.  Works for Two's Com-
       pliment values.
